
# AI Chatbot Podcast Website

This is a fully static multi‑page website ready to deploy on **GitHub Pages**.

## How to Deploy

1. Create a new GitHub repository.
2. Upload all the files in this folder (index.html, about.html, team.html, styles.css, script.js).
3. Go to:
   **Settings → Pages → Build and Deployment**
4. Set:
   - Source: **Deploy from branch**
   - Branch: **main** (or master)  
   - Folder: **Root**
5. Save and wait 10–30 seconds.
6. Your site will appear at:
   `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

## Pages Included
- `index.html` — Home / Podcast Episode
- `about.html` — About the Project
- `team.html` — Meet the Team
- `styles.css` — Stylesheet
- `script.js` — Optional JS

Fully compatible with GitHub Pages.
